Версия: 1.03
Дата 12.11.2018
Назначение: Цифровой переприем в блоке БВП.
Микроконтроллер: ATmega164

Протокол общения по ЦПП изменен на Modbus.
	Адрес устройства: 1. 
	Скорость передачи данных: 115200 бит/с.
	Количество бит данных: 8.
	Соличество стоп бит: 1.
	Контрроль четности: нет.
		
	Holding Registers (0x03, 0x06, 0x10):
		Команды для передачи. Установленный бит означает наличие команды.
		1 -> команды с 16 по 1. 
		2 -> команды с 32 по 17. 
	Input Registers (0x04):
		Команды на приеме. Установленный бит означает наличие команды.
		1 -> команды с 16 по 1. 
		2 -> команды с 32 по 17.
		
//---ПРИМЕР КОМАНД ДЛЯ РАБОТЫ С ПРОТОКОЛОМ--------------------------------------		

Запись команд для передачи:
	Запрос	: 01 06 00 00 00 01 48 0A
			  		|     |     | CRC		
			        |     | команды, в данном случае 1 команда
			        | адрес регистра, в данном сучае для команд с 16 по 1
	Ответ	: 01 06 00 00 00 01 48 0A (повторяет запрос)
	
	Запрос	: 01 06 00 00 02 01 49 6A
			  		|     |     | CRC		
			        |     | команды, в данном случае команды 1 и 9
			        | адрес регистра, в данном сучае для команд с 16 по 1
	Ответ	: 01 06 00 00 02 01 49 6A (повторяет запрос)
	
	Запрос	: 01 06 00 01 80 00 B9 CA
					|     |     | CRC		
			        |     | команды, в данном случае 32 команда
			        | адрес регистра, в данном сучае для команд с 32 по 17
	Ответ	: 01 06 00 01 80 00 B9 CA (повторяет запрос)

	Запрос	: 01 10 00 00 00 02 04 10 00 00 10 F6 A3
			        |     |     |  |     |     | CRC
			        |     |     |  |     | команды с 32 по 17, в данном случае 21
			        |     |     |  |команды с 16 по 1, в данном случае 16
			        |     |     | количество байт данных для передачи  
			        |     | количество адресов для передачи
			        |адрес первого регистра
	Ответ	: 01 10 00 00 00 02 41 C8 (повторяет заголовок запроса).                    	 
		
Считывание команд для передачи:
	Запрос	: 01 03 00 00 00 02 C4 0B
	Ответ	: 01 03 XX XX YY YY cr cr	
			        |     |     | CRC		
			        |     | команды с 32 по 17
			        | команды с 16 по 1
				        
Считывание команд на приеме:
	Запрос	: 01 04 00 00 00 02 71 CB 
	Ответ	: 01 04 XX XX YY YY cr cr	
			        |     |     | CRC		
			        |     | команды с 32 по 17
